Chronological catalog of spaceflight milestones for software version naming.
Each entry is verified against NASA, ESA, Roscosmos, and CNSA primary sources.

## Usage Guide

Suggested version naming pattern:

```
v<major>.<minor>.<patch>-<codename>
```

Examples:

- `v0.1.0-sputnik` -- the very beginning
- `v0.5.0-gemini` -- when two systems dock together
- `v1.0.0-eagle` -- "the Eagle has landed" (first stable release)
- `v2.0.0-voyager` -- a grand tour of new capabilities
- `v3.0.0-artemis` -- the return, rebuilt from the ground up

Progression follows the chronological order above: each major version advances through the timeline of human ambition in space.
