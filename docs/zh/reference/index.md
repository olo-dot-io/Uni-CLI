# 参考索引

参考页描述的是稳定合同：格式、步骤、退出码、维护命令、插件和发布规则。

- [站点目录](/zh/reference/sites)：当前可以操作哪些站点和工具。
- [适配器格式](/zh/ADAPTER-FORMAT)：YAML/TypeScript adapter 的规范。
- [管线步骤](/zh/reference/pipeline)：pipeline step 的行为和示例。
- [退出码](/zh/reference/exit-codes)：CLI 退出码和智能体处理方式。
- [维护工具](/zh/reference/maintenance)：目录、schema、统计和健康检查命令。
- [插件开发](/zh/PLUGIN)：如何扩展 Uni-CLI。
- [发布](/zh/reference/release)：版本和发布门禁。

如果你在写 adapter，先看 [适配器格式](/zh/ADAPTER-FORMAT)，再查 [管线步骤](/zh/reference/pipeline)。
